# ClassTable Plugin

Miao-Yunzai 机器人课程表插件，支持 WakeUp 课程表导入。

## 功能特性

- 通过 WakeUp 课程表 APP 的分享口令导入课表
- 自动撤回分享口令保护隐私

## 使用方法

### 导入课表

1. 打开 WakeUp 课程表 APP
2. 点击右上角分享按钮
3. 复制完整的分享口令消息
4. 将分享口令消息发送给机器人（群聊或私聊）
5. 机器人会自动识别并导入课表

### 查看课表

- `课表插件帮助` - 显示使用菜单
- `群友在上什么课` - 渲染有什么群友在上课

## 配置说明

插件会自动在 `./plugins/classtable/data/` 目录下创建课表数据文件

## 注意事项

- 开学日期默认为 2025-09-01，可在代码中修改 `SEMESTER_START_DATE` 常量
- 重复导入会覆盖之前的课表数据
